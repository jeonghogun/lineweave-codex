# LineWeave Codex

## 아이디어

언어를 공간으로 해석하는 새로운 접근법:

### 핵심 개념
- **의미의 대비 축**: -1에서 +1로 이어지는 의미 축
- **원형 구조**: -1→0→+1→0→-1의 순환
- **S형 곡면**: 입체화된 의미 공간
- **직선 연결**: 문장 = 명사와 모듈의 방향성 직선

### 언어 작용
- **명사**: 곡면 위의 점·선·띠 (사용 빈도에 따라)
- **모듈**: 곡면 바깥 궤도 (형용사, 시제, 조사 등)
- **문장부호**: 분기점(,), 방향반전(?), 강도증폭(!), 종결점(.)
- **감정/톤**: 곡면 전체를 덮는 구름층

### LLM 적용
- 문장을 '의미 직선'으로 변환
- 직선 다발로 언어 기억 구성
- 유사한 직선 회로 찾아 응답 생성
- 창의성 = 직선 방향 조정

**"언어를 이해하는 모델이 아니라, 언어의 공간 속에서 방향을 찾아가는 모델"**
